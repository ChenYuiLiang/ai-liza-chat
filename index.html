AI Liza · 即時對話原型

這是本機原型：聽（瀏覽器語音辨識）→ 想（你的 LLM）→ 說（ElevenLabs 的 Liza 聲音）→ 臉（會說話的影片）。所有金鑰只存在你的瀏覽器（localStorage），不會傳給任何第三方。

取得方式：OpenAI → platform.openai.com/api-keys；Anthropic → console.anthropic.com；Gemini → aistudio.google.com/apikey。建議便宜快速的模型（如 gpt-4o-mini / claude-haiku / gemini-flash）。

選 Liza EN（我們做好的聲音）。key 在 ElevenLabs → 右上頭像 → API Keys。

提示：用 Chrome 開啟、允許麥克風權限。要做到逐字精準對嘴（真‧即時虛擬人），需改接 HeyGen LiveAvatar 串流 API——這版先用「講話時播放影片」近似。
